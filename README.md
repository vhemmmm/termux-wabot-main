
## Clone this project

```bash
> git clone https://github.com/vhemmmm/termux-wabot-main
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg install unzip
> unzip termux-wabot-main
> cd termux-wabot-main
> bash install.sh
```

### Usage
```bash
> npm start
```
